<h1 align=center>Vitra - CustomTkinter Project - Test</h1>

<p align=center>This project was created with the goal of experimenting and learning about CustomTkinter, exploring its capabilities to create modern graphical interfaces in Python. Additionally, it is used to practice code modularization, breaking functionalities into separate modules to improve organization and scalability. Through this approach, the goal is to understand how to design attractive and well-structured applications while deepening knowledge of CustomTkinter and good programming practices.</p>

### **Installation**

Clone the repository:

```bash
git clone https://github.com/LaureanoCz/vitra.git
```
Install the dependencies and run the project:

```bash
pip install -r requirements.txt
python main.py
```
